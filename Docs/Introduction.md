# Introducción
El Sistema Andon tiene como objetivo monitorear en tiempo real el estado de las líneas de producción, permitiendo registrar fallas, operadores, modelos y notificar incidencias vía Telegram.

## Tecnologías
- Backend: .NET 8 (C#)
- Frontend: React + Tailwind
- Base de datos: PostgreSQL
- Integración: Telegram Bot API
